# React Native Reels (Expo + TypeScript)

A full-screen vertical **Reels / TikTok style** screen built using **Expo**, **Expo Router**, **TypeScript**, and **expo-av**.

Each reel supports:
- Video (auto-play)
- Image
- Gradient background
- Per-reel music (auto-plays for active reel)
- Music stops on swipe + back navigation

---

## Installation

```bash
npm install
expo install expo-av expo-linear-gradient
expo start -c
```


For running project in real device you can install the **Expo Go** app and then scan QR code that is showing in terminal while executing project.
